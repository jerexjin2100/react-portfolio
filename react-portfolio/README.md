# React Portfolio

A personal portfolio website built with React and Tailwind CSS to showcase projects, skills, and contact information.

## Features
- Multi-page routing using React Router
- Responsive dark-themed UI
- Reusable component-based architecture
- Projects showcase with live and GitHub links
- Contact form layout

## Tech Stack
- React
- Tailwind CSS
- JavaScript
- Vite
